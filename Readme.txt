EEE4022S: Development of an interactive badge for classrooms

The aim of this project is to develop an interactive badge that enables device to device
connections and plays an effective role in participation in classrooms.

Main files:
StudentDevice - contains the overall student device code
TeacherDevice - contains the overall teacher device code
AnswerAnalysis.cpp and AnswerAnalysis.h - libraries created for storage

Other files:
All other files are the subsystems for each device. 

In order to use the code, you will be required to download the ESP32 board
from expressify. 

Install all the libraries mentioned in each code file.
